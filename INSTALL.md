# تركيب النظام — PrintShop Local

## المتطلبات الأساسية
- Windows 10 أو أحدث
- Python 3.8+
- طابعة متصلة بالحاسوب
- مودم WiFi (موجه) لتوزيع الشبكة

## خطوات التركيب

### 1. تثبيت Python
تأكد من تثبيت Python 3.8+ مع pip.

```bash
python --version
pip --version
```

### 2. تشغيل سكريبت التثبيت
```bash
setup.bat
```

هذا سيقوم تلقائياً بـ:
- تثبيت جميع المكتبات المطلوبة
- تهيئة قاعدة البيانات
- إنشاء الحسابات الافتراضية
- توليد أكواد QR
- توليد شعار المحل

### 3. إعداد الشبكة (WiFi)

#### الخيار A — استخدام موجه WiFi منفصل (موصى به)
1. اشترِ موجه WiFi رخيص (~15$)
2. اعدل إعداداته:
   - SSID: `LAIDANI_PRINT`
   - Password: `laidani2024`
   - LAN IP: `192.168.1.1`
   - DHCP: تشغيل (المدى 192.168.1.100 - 192.168.1.200)
3. وصّل كل كمبيوتر بالسلك أو WiFi بالموجه

#### الخيار B — استخدام Windows Hotspot
شغّل `captive_portal_setup.bat` كمسؤول واتبع التعليمات.

### 4. تشغيل النظام
```bash
start.bat
```

### 5. فتح لوحة التحكم
- **الزبون:** `http://192.168.1.1:5000/upload/PC1`
- **العامل:** `http://192.168.1.1:5000/worker/login`
- **المدير:** `http://192.168.1.1:5000/manager/dashboard`

### 6. طباعة أكواد QR
- توجد في مجلد `qrcodes/`
- اطبع كل QR Code وضعه أمام الكمبيوتر المخصص له

## حل المشاكل الشائعة

### "لا يمكن الاتصال بالخادم"
- تأكد أن السيرفر شغال
- تأكد من رقم IP في `config.py`
- تأكد من جدار الحماية (قد يحجب المنفذ 5000)

### "الطباعة لا تعمل"
- تأكد من توصيل الطابعة
- شغّل `python -c "from printer import get_available_printers; print(get_available_printers())"`
- قدّم اسم الطابعة الصحيح

### "الملفات لا ترفع"
- تأكد من حجم الملف (الحد الأقصى 50MB)
- الصيغ المدعومة: PDF, JPG, PNG, DOCX, XLSX, PPTX
